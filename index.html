⚙ Configuración Inicial

Conecta el dashboard con tu Google Sheets. Solo se hace una vez.

Paso 1

Ve a sheets.google.com → crea una hoja nueva → nómbrala Camacho's PRO

Paso 2

En el Sheet: Extensiones → Apps Script → borra el contenido → pega el archivo camacho_apps_script.js → guarda (Ctrl+S)

Paso 3

Recarga el Sheet → menú 🏆 Camacho's PRO → Inicializar Sistema → autoriza cuando te pida permiso

Paso 4

En Apps Script: Implementar → Nueva implementación → Tipo: Aplicación web → Acceso: Cualquier persona → Implementar → Copia la URL

Paso 5

Pega la URL aquí abajo y conecta.

Comienza con https://script.google.com/macros/s/
